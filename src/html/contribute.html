There are so many ways in which you can help us grow Tech Roadmap into a budding tech community and the no. 1 online resource for tech careers and learning. You can do so by

  • adding a new roadmap,
  • editing, or suggesting changes to the existing roadmaps,
  • contributing to the content and resources of the website,

  • contributing to the UI design of the site,
  • refactoring our codebase, etc.

If you want to contribute as a developer, you can do so here. If you are a non-developer or do not know how to code but still want to contribute, you can do that here.

Do read our contribution guidelines first before making any contributions and please give our Github repo a star, it would mean much to us.

This project is OpenSource and is hosted on GitHub. All meaningful contributions are welcome to help grow this project and you don't need to know how to code to contribute.

If you would like to contribute to this project, please visit our contributors page to find out how to contribute.

A considerable amount of my time is spent doing unpaid community work on things that I hope will help humanity in some way. Your sponsorship helps me continue to produce more open-source and free educational material consumed by hundreds of thousands of developers every month.
