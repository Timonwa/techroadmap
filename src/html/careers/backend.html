What is Backend Development?

Backend development, also called server-side development, is the process of creating everything that the users don't see and contains behind-the-scenes activities that occur when performing any action on a website. It focuses primarily on databases, backend logic, APIs, and Servers. Backend development can be done in almost any programming language such as JavaScript (using the NodeJS runtime engine), Python, C, C# and C++, PHP, etc.

Databases

A database is an organized collection of structured information, or data, typically stored electronically in a computer system. There are different types of databases and they can all be integrated into a Node.js application.

MySQL

MySQL is one of the more popular database platforms that features a Relational Database Management System and used specially constructed instructions called queries to perform actions on data.

  • MySQL Tutorial for Beginners ~ Programming with Mosh Video

MongoDB

MongoDB is a popular NoSQL database platform. This means that instead of queries, it uses a Document Data Model, much like JSON, that allows for faster, more flexible application development.

Javascript Frameworks for Backend Development

A JavaScript framework is an application framework written in JavaScript where programmers can manipulate the functions and use them for their convenience. They make working with JavaScript easier and smoother. So you have to have reasonable knowledge of JavaScript before using it.

Express Js

Express is a minimal and flexible Node.js web application framework that provides a robust set of features for web and mobile applications.

With a myriad of HTTP utility methods and middleware at your disposal, creating a robust API is quick and easy.

Express provides a thin layer of fundamental web application features, without obscuring Node.js features that you know and love.
